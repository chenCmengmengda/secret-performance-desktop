#### 介绍
secret-performance-desktop(桌面性能监控)

#### 功能
* cpu和内存的实时监控
* 开机自动启动
* 自定义logo
* 自定义主题颜色
* 鼠标拖拽
* 设置纵向或横向布局
* 桌面时钟
* 黑客帝国代码雨
* 雪花飘落
* 樱花飘落
